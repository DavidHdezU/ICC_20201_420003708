¿Cómo usar el programa?

1.- Abrir una ventana en la terminal
2.- Dirigirse a la carpeta practica06/
3.- En la terminal escribir "javac src/Matrix.java" para compilar la clase Matrix
3.- En la terminal escribir "javac src/MatrixTest.java" para compilar la clase MatrixTest donde están las pruebas para ver si los métodos funcionan correctamente
4-. Después de que haya compilado escribimos en la terminal "java src/MatrixTest.java" para correr nuestro programa. Cabe mencionar que en la clase MatrixTest se encuentra todo el codigo necesario para que la practica salga como se espere.
5.- Si no hubo errores en la terminal debería haber impreso que todos los métodos fueron exitosos (<metodo> succesfull :D)
6.- La practica debería haber salido bien
