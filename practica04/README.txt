¿Cómo usar el programa?

1.- Abrir una ventana en la terminal
2.- Dirigirse a la carpeta src/
3.- En la terminal escribir "javac main/Main.java" para compilar la clase Main, lo cual debería compilar también todas las clases dentro de la carpeta shapesSVG/.
4-. Después de que haya compilado escribimos en la terminal "java main/Main.java" para correr nuestro programa. Cabe mencionar que en la clase Main se encuentra todo el codigo necesario para que la practica salga como se espere.
4.- Si no hubo errores en la terminal debería haber impreso lo que se especificaba en la clase Main
5.- Después en la terminal escribir "java main/Main.java > main.html", lo cual no generará dentro de la carpeta src/ un archivo .html (main.html) el cual al ejecutarlo debería abrir una ventana, en el navegador que tengamos predeterminado, en la cual se puedan ver todas las figuras.
6.- La practica debería haber salido bien
